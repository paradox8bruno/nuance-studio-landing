# Legacy Backup

Backup dos arquivos estáticos anteriores à migração para Next.js.

- `html/index.html`: antiga página de vendas estática.
- `html/quiz-index.html`: antigo quiz estático.
- `html/politica-de-privacidade.html`: antiga página legal estática.
- `html/termos-de-uso.html`: antiga página legal estática.
- `html/404.html`: antiga página 404 estática.
- `api/meta-events.js`: antiga função serverless da Meta Conversions API.

As rotas ativas agora vivem em `app/` e os assets públicos oficiais ficam em
`public/fotos/`.
